# Snake Gun Water Game 🎮

A simple Python command-line game where the user plays Snake, Gun, Water against the computer.

---

## 📌 About the Project
This is a beginner-friendly Python project based on a fun game similar to Rock Paper Scissors.

The user plays against the computer and the winner is decided based on game rules.

---

## 🧠 Rules
- Snake 🐍 drinks Water 💧 → Snake wins
- Gun 🔫 kills Snake 🐍 → Gun wins
- Water 💧 damages Gun 🔫 → Water wins

---

## 🚀 How to Run
1. Make sure Python is installed
2. Open terminal / command prompt
3. Run the file using:

```bash
python game.py
